Forms

Forms allow users to fill in data from a browser and submit it to the server, where it gets process and sends a response for the browser to display.

creating a form

  • action = "destination you want" tells the broswer what program to send the form data to, e.g. :
    • form action="http://www.server.com/subdirectory/program.php"
    • form action="subdirectory/program.php" (if the program is on the same server as the html file)
    • form action="program.php" ( if the program is in the same directory as the html file )
  • method can be either "get" or "post"
    • method= "get" get is the defaul method, the form data will be displayed on the URL (like search query in seach engine), but can only handle small transmission (few hundred characters) and cannot keep secret
    • method = "post" you can't see any data when using post method, post can also handle a big transmission (a file)
  • form textarea example:

    Please enter any feedback below.

  • a submit button is needed for user to submit the data to the server.

Other attribues in plain text field (type="text") :

  • value="something" will be what is shown what page is loaded
  • placeholder="something" will be shown initially but disappear when user enter data
  • autofocus sets which field is highlighted
  • required means this field must be filled, else submission declined

Additionally, label can be added to the input to assign that input field to a label so that when that label is clicked, the field will highlights

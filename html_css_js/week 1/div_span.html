positioning an element

Many element, including "div" can be put at specified position by:

  • Using position:absolute; top:XXpx; left:YYpx to specify top left corner of element relative to the page.
  • Using position:relative; top:XXpx; left:YYpx to specify top left corner of element relative to where it would normaly appear.

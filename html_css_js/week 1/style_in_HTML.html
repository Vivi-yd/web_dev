  • colour for text colour
  • background for background colour
  • font-family for text fonts
  • font-size for text sizes
  • text-align for text alignment

Psudo Classes

Are classes with some kind of "intelegence", the style can be set to change after some conditions are fulfilled, below are some examples:
  • Hover - text change colour when cursor is over it
  • a: link - a link
  • a: visited - a link that has been visited
  • a: active - means a link is currently processing
  • p: empty - means an empty element
